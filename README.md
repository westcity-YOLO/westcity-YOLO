# 📦 Core Capabilities

### 🧩 1｜嵌入式系统与端侧 AI
- 基于 **ESP32 / Tuya T5** 开发物联网设备，精通 Wi-Fi / BLE / Zigbee 无线通信协议
- 使用 **RT-Thread** 实现多线程调度、设备驱动开发与系统裁剪优化
- 通过 **LVGL / Qt** 完成资源受限平台的嵌入式 GUI 交互设计
- 利用 **Edge Impulse** 搭建完整 TinyML 流程，实现关键词识别、异常检测等端侧智能

### 🖥 2｜边缘计算与 Linux 开发
- 平台适配：**Raspberry Pi、Rockchip、Allwinner、K230** 等边缘计算硬件
- Linux 开发：设备树、驱动调试、交叉编译、Buildroot 系统构建
- 视觉与音频：**YOLO** 目标检测、OpenCV 图像处理、FFmpeg 音频处理
- 推理部署：TensorFlow Lite / RKNN / ONNX Runtime 模型量化与加速

### 🚀 3｜全栈开发与云服务
- 前端：**Vue3** 后台管理系统、响应式布局与组件化开发
- 跨端：**微信小程序** 实现设备控制、数据展示与云开发对接
- 后端：**Flask** 构建 RESTful API、物联网数据平台与权限鉴权
- 数据库：MySQL / SQLite 库表设计、索引优化与高并发处理

### 🔌 4｜硬件设计与结构建模
- **KiCad / 嘉立创 EDA** 完成原理图设计、PCB Layout、打样与调试
- **FreeCAD / Blender** 进行产品 3D 结构建模、外观设计与结构验证
