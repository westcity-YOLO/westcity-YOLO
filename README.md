# 📦 Core Capabilities

## 1｜嵌入式系统与端侧 AI
- 基于 **ESP32 / 涂鸦T5** 的物联网设备开发，熟悉 Wi-Fi / BLE / Zigbee 无线通信
- **RT-Thread** 多线程管理、驱动开发、系统裁剪与性能优化
- **LVGL / Qt** 嵌入式 GUI 设计与资源受限平台适配
- **Edge Impulse** 端侧机器学习：传感器采集、模型训练、嵌入式部署

## 2｜边缘计算与 Linux 开发
- 适配 **树莓派 / RK35xx / 全志 / K230** 等边缘计算平台
- Linux 应用开发、设备树、驱动调试、Buildroot 系统构建
- **YOLO / 深度学习** 模型落地，图像 / 音频处理与端侧推理优化
- 支持 TensorFlow Lite / RKNN / Tengine 等推理框架部署

## 3｜全栈开发与云服务
- 前端：**Vue3 + ES6+** 后台管理系统、响应式开发
- 跨端：**微信小程序** 设备控制、数据展示、云开发对接
- 后端：**Flask** 构建 RESTful API、物联网数据平台与权限管理
- 数据库：MySQL / SQLite 设计、索引优化与事务处理

## 4｜硬件设计与结构建模
- **KiCad / 嘉立创 EDA** 完成原理图、PCB Layout、打样与调试
- **FreeCAD / Blender** 3D 结构建模、产品外观与结构验证
